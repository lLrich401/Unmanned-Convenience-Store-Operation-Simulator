  ## 게임 목표                                                                                                                                                                                                      
                                                                                                                                                                                                                    
  초기 자금은 50,000원입니다.                                                                                                                                                                                       
                                                                                                                                                                                                                    
  - 보유 자금이 120,000원 이상이 되면 성공 엔딩입니다.                                                                                                                                                              
  - 보유 자금이 10,000원 미만이 되면 파산 엔딩입니다.                                                                                                                                                               
                                                                                                                                                                                                                    
  ## 주요 기능                                                                                                                                                                                                      
                                                                                                                                                                                                                    
  - 상품 재고 확인                                                                                                                                                                                                  
  - 상품 가격 조정                                                                                                                                                                                                  
  - 재고 보충                                                                                                                                                                                                       
  - 홍보 진행                                                                                                                                                                                                       
  - 보안 강화                                                                                                                                                                                                       
  - 하루 영업 시뮬레이션                                                                                                                                                                                            
  - 매출, 도난, 할인, 운영비 로그 확인                                                                                                                                                                              
                                                                                                                                                                                                                    
  ## 조작 방법                                                                                                                                                                                                      
                                                                                                                                                                                                                    
  - 마우스 클릭: 버튼 선택                                                                                                                                                                                          
  - SPACE: 하루 영업 시작                                                                                                                                                                                           
  - R: 새 게임                                                                                                                                                                                                      
  - ESC: 게임 종료                                                                                                                                                                                                  
                                                                                                                                                                                                                    
  ## 화면 구성                                                                                                                                                                                                      
                                                                                                                                                                                                                    
  - 상단 상태 패널: 자금, 평판, 홍보 점수, 보안 점수 표시                                                                                                                                                           
  - 상품 관리: 상품별 가격과 재고 조정                                                                                                                                                                              
  - 운영 행동: 홍보, 보안 강화, 영업 시작                                                                                                                                                                           
  - 매장 화면: 매장 상태를 그래픽으로 표시                                                                                                                                                                          
  - 운영 로그: 영업 결과와 이벤트 표시                                                                                                                                                                              
                                                                                                                                                                                                                    
  ## 사용 기술                                                                                                                                                                                                      
                                                                                                                                                                                                                    
  - Python                                                                                                                                                                                                          
  - pygame
